# DS Review

Personal repo for Python review and interview prep — covering fundamentals, OOP, and (upcoming) data structures & algorithms.

## Structure

```
python_review/     # Python fundamentals: variables, strings, lists, dicts,
                   # sets, loops, conditionals, functions, exceptions, etc.

py_oop/            # Object-oriented programming
├── class_obj/     #   Classes, objects, attributes, methods, __init__, self
├── class_attributes/  #   Class vs instance attrs, class methods, static methods
├── encapsulation/ #   Access control, properties, getters/setters, decorators
├── inheritance/   #   Single/multi inheritance, super(), MRO, overriding
├── polymorphism/  #   Duck typing, method overloading, inheritance-based polymorphism
└── abstraction/   #   Abstract classes, interfaces, concrete implementations
```
